This website presents an end-to-end analysis workflow for detecting marker genes using the jazzPanda R package. Additional analysis scripts and extended explorations used in the jazzPanda paper are available in the analysis repository. The underlying datasets, including raw vendor data and lightly processed data objects, are publicly available via Zenodo.

This website hosts the analysis outputs for detecting marker genes with the .

jazzPanda is designed for spatial transcriptomics data and integrates spatial context into marker detection, providing more robust identification of genes that define clusters of cells across different tissue contexts. The analyses presented here demonstrate the application of jazzPanda to a variety of publicly available datasets from multiple spatial technologies.
